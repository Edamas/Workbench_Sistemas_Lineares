# Workbench de Sistemas Lineares

Esta é uma aplicação web interativa construída com Streamlit para resolver e analisar sistemas de equações lineares.

## Funcionalidades

*   **Entrada de Equações:** Insira sistemas de equações em texto plano.
*   **Análise de Matrizes:** Visualize a matriz aumentada e sua forma escalonada e escalonada reduzida.
*   **Operações de Linha:** Aplique operações elementares de linha passo a passo.
*   **Classificação do Sistema:** Identifica se o sistema é Possível e Determinado (SPD), Possível e Indeterminado (SPI) ou Impossível (SI).
*   **Visualização:** Suporta notação em LaTeX e texto plano.

## Como Executar

1.  **Instale as dependências:**
    ```bash
    pip install -r requirements.txt
    ```

2.  **Execute a aplicação:**
    ```bash
    streamlit run app.py
    ```
